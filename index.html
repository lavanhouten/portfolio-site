www.Top25CollegeFootball.com

During this project, I used AWS to create a website that has a Flask backend server that scrapes data from across the internet, stores it in a DynamoDB database, and then ranks teams using that data using Sci Kit Learn Multilayer Perceptron and Logistic Regression models. It also uses a GraphQL connection to connect the frontend to the database. Other APIs are also used to gather news and top Youtube channels. Currently the backend server models and database configuration are undergoing a revamp in preparation for the upcoming season.
